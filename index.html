Bem-vindo ao projeto JRimum

JRimum é um projeto open source focado no desenvolvimento de componentes de software para o domínio de negócios do Brasil.

Fruto da iniciativa conjunta entre estudantes do IFRN e a empresa Nordeste Fomento Mercantil, o JRimum teve como sua primeira missão a criação de um componente Java para a geração de boletos bancários, o Bopepo


Devido ao sucesso do design reusável para assuntos recorrentes (como objetos de domínio bancário), o projeto cresceu e focou seu desenvolvimento em mais componentes, também relacionados a outros domínios.


Atualmente o projeto JRimum é composto por 5 sub-projetos em diferentes status de desenvolvimento:

  • Bopepo — Gerador de boletos bancários personalizados [Lançado individualmente, incubado]
  • Texgit — Manipulador de strings e objetos orientados por layout (ex: arquivos texto bancários) [Protótipo Lançado, incubado]
  • Domkee — Entidades e tipos comuns a diferentes domínios [Lançado embutido, incubado]
  • Utilix — Conjuntos de utilitários utilizados no projeto e de uso geral [Lançado embutido, incubado]
  • Vallia — Validadores para diversos tipos e entidades de domínios comuns [Lançado embutido, incubado]

A cada dia que passa, a lista de casos de sucesso cresce e várias empresas se desenvolvem ao agregar os componentes do projeto em produção.
